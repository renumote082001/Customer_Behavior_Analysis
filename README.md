📊 Project Overview

This project focuses on analyzing customer shopping behavior using transactional data. The goal is to identify key patterns in customer spending, product preferences, and subscription behavior to help businesses optimize marketing strategies and improve revenue.

🎯 Objectives
-Understand customer purchase behavior
-Identify high-value customers and segments
-Analyze impact of discounts and subscriptions
-Discover top-performing products
-Provide actionable business recommendations
📁 Dataset Summary
-Total Records: 3,900 transactions
-Features: 18 columns
Includes:
-Customer demographics (Age, Gender, Location)
-Purchase details (Product, Category, Amount)
-Behavior data (Discounts, Frequency, Ratings, Shipping)
🛠️ Tools & Technologies
-Python (Pandas, NumPy) – Data cleaning & preprocessing
 -MySQL – Business analysis
Power BI – Dashboard & visualization
⚙️ Project Workflow
🔹 1. Data Cleaning & Preparation (Python)
-Handled missing values (Review Ratings using median)
-Standardized column names (snake_case)
-Created new features:
-age_group
-purchase_frequency_days
-Removed redundant columns (promo_code_used)
🔹 2. Data Analysis (SQL)
-Performed business-driven analysis:
-Revenue by Gender
-High-spending discount users
-Top-rated products
-Shipping type comparison
-Subscribers vs Non-subscribers
-Discount-dependent products
-Customer segmentation (New, Returning, Loyal)
-Top products per category
-Repeat buyers vs subscriptions
-Revenue by age group
🔹 3. Dashboard (Power BI)
Created an interactive dashboard to visualize:
-Customer segments
-Revenue trends
-Product performance
-Purchase behavior patterns
📊 Key Insights
-Discount users can still be high spenders
-Loyal customers contribute significantly to revenue
-Subscription users show higher engagement
-Certain products rely heavily on discounts
-Age group impacts spending patterns
💡 Business Recommendations
-Increase subscription benefits to boost retention
-Implement customer loyalty programs
-Optimize discount strategy to protect margins
-Promote top-rated products in campaigns
-Use targeted marketing for high-value segments
📈 Outcomes
-Built end-to-end data analysis pipeline
-Improved understanding of customer behavior
-Delivered actionable insights for business growth
